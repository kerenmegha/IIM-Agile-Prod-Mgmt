# Task Manager

A simple one-page task manager built with HTML, CSS, and JavaScript.

## Features

- Add, complete, and delete tasks
- Set optional due dates when creating tasks
- Monthly calendar with task indicators on each day
- Click a date to filter tasks due that day
- Overdue tasks are highlighted in red
- Filter by all, active, or completed
- Voice input via microphone (Chrome and Edge)
- Clear all completed tasks at once
- Tasks persist in your browser via localStorage

## Run locally

Open `index.html` in your browser, or serve the folder with any static file server:

```bash
npx serve .
```
